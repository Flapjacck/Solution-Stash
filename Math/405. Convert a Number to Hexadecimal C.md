# [405. Convert a Number to Hexadecimal](https://leetcode.com/problems/convert-a-number-to-hexadecimal/description/)

## [Solution Link](https://leetcode.com/submissions/detail/1872337159/)

## Notes

This solution converts an integer to its hexadecimal representation:

1. **Hexadecimal conversion**: The `%x` format specifier in `sprintf` handles the conversion from decimal to hexadecimal automatically
2. **Character array**: Allocate 10 characters (sufficient for the hexadecimal representation of a 32-bit integer)
3. **Built-in function**: Leverage standard library function to avoid manual bit manipulation or division operations

The key insight is using `sprintf` with the `%x` format specifier, which is a simple and efficient approach compared to manual digit-by-digit conversion.

### Time Complexity: O(log n)

- The `sprintf` function iterates through each hexadecimal digit
- Number of hexadecimal digits in an integer is log₁₆(n)
- Since log₁₆(n) = log₁₀(n) / log₁₀(16), this is still O(log n)
- For a 32-bit integer, maximum 8 hexadecimal digits are possible
- Overall: O(log n) where n is the input integer

### Space Complexity: O(1)

- Fixed allocation of 10 characters regardless of input size
- A 32-bit signed integer requires at most 8 hexadecimal digits plus null terminator
- Space usage doesn't depend on input size (constant space)

```c
char* toHex(int num) {
    char *p = (char*)malloc(sizeof(char)*10);
    sprintf(p, "%x", num);
    return p;
}
```
